backing up saved chrome passwords

this is going to be a pretty short one because i'm mostly writing this for my own benefit. if you have a situation where someone can't/won't back up their passwords against a google account, this method can be used to sling the passwords over to a new machine:

  1. on the source machine, open Chrome, go to Settings > Passwords, and use the Export option from the three dot menu. This will give you a CSV file with all the passwords in plaintext.
  2. On the destination machine, open Chrome and enter chrome://flags into the navigation bar. enable the password import flag, and relaunch Chrome
  3. Once relaunched, go to Settings > Passwords, and use the Import option from the three dot menu. Select the CSV file from step one, and you're done.
