youtube-dl YouTube playlist podcast downloader:

I've recently had the want to download an entire YouTube playlist as audio files so that I can listen to them as a podcast. I figured the easiest way to do this would be with youtube-dl.

I had a few things I wanted to accomplish with this:

  1. Save each video as its own .mp3 file
  2. Embed track number, artist, album, and title metadata into mp3 file
  3. Embed YouTube thumbnail as album art into the mp3 file

Below is the command I came up with:

youtube-dl \
--extract-audio \
--audio-format mp3 \
--audio-quality 0 \
--embed-thumbnail \
-o "%(playlist_index)s %(title)s.%(ext)s" \
--postprocessor-args "-metadata album='ALBUM' -metadata artist='ARTIST'" \
"https://www.youtube.com/watch?v=PLAYLIST"

One thing that I had issues with was embedding the track number (pulled from where in the playlist the video is) into the mp3 metadata, instead I have added it to the beginning of the name of each file - this doesn't bother me that much. I've also chosen to specify the album and artist metadata manually - I didn't like the results I got when I extracted these from the YouTube video.

Here's the entire command in one nice, copy-pastable chunk:

youtube-dl --extract-audio --audio-format mp3 --audio-quality 0 --embed-thumbnail -o "%(playlist_index)s %(title)s.%(ext)s" --postprocessor-args "-metadata album='ALBUM' -metadata artist='ARTIST'" "https://www.youtube.com/watch?v=PLAYLIST"
