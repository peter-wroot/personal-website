Upgrading my Mac Mini Server

I've been using a 2012 Mac Mini as my main home server for a little over 6 months now, and up until recently I've been very happy with how it performs. However, in the last few weeks the cracks have started to show: as it turns out, MacOS isn't really a great server operating system.

I recently stopped using iCloud keychain and moved to using Bitwarden for my password management needs
